# vless-proxy-extension
Расширение для браузеров Chromium, которое устанавливает соединение с десктоп-прокси 
<hr>
Как добавить в браузер

1 - Откройте Chrome и перейдите на страницу «Расширения», введя chrome://extensions/ в адресной строке. Включите режим разработчика:

2 - Переключите переключатель «Режим разработчика» в правом верхнем углу. Загрузите расширение:

3 - Нажмите «Упаковать расширение» и выберите папку, содержащую расширение, в которой находится файл manifest.json.

4 - Нажмите «Загрузить распакованное» и выберите папку

<hr>

Как настроить v2rayN клиент

1 - Добавить свой клиент <br>

2 - Установить в claer system proxy <br>

3 - В настройках Mixed Port выставить 10808 или поменять на свой в background.js в 28 строчке <br>

После нажать на расширение и нажать enable proxy
